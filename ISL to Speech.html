Help & Guide

Follow the steps to use this tool:

  1. Allow camera access.
  2. Make a sign gesture in front of the camera.
  3. Click "Capture Gesture" to process the sign.
  4. The recognized text will appear, and you can convert it to speech.

About This Project

This project aims to bridge communication gaps by converting sign language into speech using AI-powered recognition and text-to-speech conversion.
